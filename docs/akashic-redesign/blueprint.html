Cosmos AI Engineering Charter v1.5

Updated2026-05-17

Visibilitypublic
Schemav2.1

The Charter is the single source of engineering truth for every AI in the Cosmos AI Lab. It encodes six Pillars and thirteen operating principles that govern how each agent reads context, plans, ships, and proves its own work — so a session run by Orion, Mythos, or Ra produces the same disciplined output against the same standard.

“One task, one branch, one PR — and a Self-Proof that a stranger could verify.”

Core mechanisms include the Self-Proof template (a seven-line attestation closing every change), the Anti-Loop rules (stop after fifteen minutes on the same failing approach, escalate rather than spin), and the Pre-Flight Gate that blocks new work on a red baseline. Version 1.5 folds in §3.8 Repo Execution Safety and the RAG-pack convention for distributing this card across the fleet.
