Cosmos AI Engineering Charter v1.5

AK-GUI-0000037
Updated 2026-05-17
Owner Mythos
Visibility Public
Schema card-v2.1

The Charter is the universal engineering contract every Cosmos AI Lab agent reads before it writes a single line. It codifies 6 Pillars and 13 principles — from Pre-Flight Gate discipline to the Single Owner Per Module rule — so that work done by Mythos, Orion, Ra or any future teammate stays coherent across a fleet of independent minds.

At its core sit two safeguards. The Self-Proof template forces each change to state, in seven lines, what it claims and how it was verified; the Anti-Loop rules halt a session the moment it spends more than fifteen minutes circling the same broken approach. Restraint over ornament — the same instinct that drives this redesign.
